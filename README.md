# chip8

## Input

    1	2	3	4

    q	w	e	r

    a	s	d	f

    z	x	c	v

## Usage

Requires SDL2

#### Linux

Install with `make`

Run with `./chip8 ROM`

#### Mac

Install with `make mac`

Run with `./chip8 ROM`
